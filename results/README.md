# Results

This folder contains portfolio-safe visual outputs generated from the notebook CSV results.

## Folders

- `figures/`: result charts for ranking, learning curves, per-class accuracy, and model complexity.
- `snapshots/`: notebook-style result table snapshots for quick portfolio viewing.
- `architecture_diagrams/`: corrected architecture diagrams for the Dense and CNN models.

## Key Files

- `figures/final_model_ranking.png`
- `figures/learning_curves.png`
- `figures/per_class_accuracy.png`
- `snapshots/final_ranking_snapshot.png`
- `snapshots/experiment_summary_snapshot.png`

The underlying numeric data is stored in the top-level `csv/` folder.
